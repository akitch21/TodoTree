# Overview

## Purpose

TodoTree は、階層構造のタスクをプロジェクト単位で管理するフルスタックWebアプリです。個人タスクも別枠で管理できます。

## Target Users

コードとREADMEから確認できる想定ユーザー:

- 個人または小規模チームでタスクを整理したいユーザー
- 親子関係を持つタスクをツリーとして見たいユーザー
- プロジェクトごとにメンバーを招待して作業したいユーザー

未確定:

- 組織単位、ワークスペース単位の管理が正式な要件かどうか
- 課金プランを本番提供する時期

## Main Use Cases

- ユーザー登録、ログイン、ログアウト
- プロジェクト作成、閲覧、更新、削除
- プロジェクトメンバー管理
- メールアドレスによるプロジェクト招待
- 招待されたユーザーによる承認
- プロジェクトタスクの作成、編集、削除、ステータス更新
- 親子関係を持つ階層タスクの表示
- 個人タスクの作成、更新、削除
- 退会

## Current Features

コードから確認できる機能:

- JWT 認証
- React Router によるSPAルーティング
- プロジェクト一覧、詳細、概要、ツリー、タスク一覧、メンバータブ
- プロジェクトメンバーの owner / admin / member ロール
- owner/admin による招待作成、招待一覧、招待取消
- 招待された本人による承認
- free plan の seat limit 判定
- PostgreSQL + Alembic migration
- Vercel / Railway 向け環境変数設定
- APIレスポンスの `Cache-Control: no-store`
- 初回登録時のオンボーディングチュートリアル（ステップ式モーダル、各ステップスキップ可）

## Current Limitations

- タスク APIはすべてプロジェクトメンバー認可チェック済みです（未認証: 401、非メンバー: 404）。
- Stripe連携用のユーザー列はありますが、課金処理そのものは未実装です。
- 通知、コメント、添付ファイル、ドラッグ&ドロップは未実装です。
- リアルタイム更新は未実装です。
