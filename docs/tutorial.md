# オンボーディングチュートリアル

## 概要

新規ユーザーが初めてログインした際に、ToDoTree の基本的な使い方をガイドするステップ式モーダルを表示します。

## 動作仕様

### 表示条件

`localStorage` のキー `todotree-onboarding` が `"pending"` のとき、アプリレイアウト上にモーダルを表示します。

| タイミング | `todotree-onboarding` の値 | 挙動 |
|---|---|---|
| 新規ユーザー登録直後 | `"pending"` にセット | 次回ダッシュボード表示時にモーダルが出る |
| チュートリアル完了 / 全スキップ | `"done"` に更新 | 以後モーダルは表示されない |
| 既存ユーザーのログイン | 変更なし | `"done"` ならモーダルは出ない |

### ステップ一覧

| # | タイトル | 内容 |
|---|---|---|
| 1 | ToDoTree へようこそ | アプリの概要説明 |
| 2 | プロジェクトを作成する | サイドバーの「プロジェクト」からプロジェクトを作成する方法 |
| 3 | タスクを追加する | プロジェクト詳細でタスク・サブタスクを追加する方法 |
| 4 | カンバンでステータスを管理する | 待機中 / 進行中 / 完了のステータス管理とカンバンビュー |
| 5 | チームメンバーを招待する | メールアドレスによるメンバー招待の方法 |

### スキップ操作

モーダルには 3 種類のスキップ手段があります。

- **このステップをスキップ** — 現在のステップを飛ばして次へ進む（最終ステップでは「完了」と同義）
- **チュートリアルをスキップ** — 残りのステップをすべて飛ばしてモーダルを閉じる
- **バックドロップクリック / × ボタン** — 「チュートリアルをスキップ」と同じ挙動

## 実装ファイル

| ファイル | 役割 |
|---|---|
| `frontend/src/store/TutorialContext.tsx` | ステップ状態・スキップロジック・ localStorage フラグ管理 |
| `frontend/src/components/tutorial/TutorialModal.tsx` | ステップ式モーダル UI |
| `frontend/src/store/AuthContext.tsx` | `register()` 内で `todotree-onboarding = "pending"` をセット |
| `frontend/src/components/layout/AppLayout.tsx` | `TutorialProvider` でラップし `TutorialModal` をレンダリング |

## 将来の拡張例

- バックエンドの `User` モデルに `onboarding_completed_at` を追加することで、複数デバイス間でチュートリアル完了状態を共有できます。
- ステップにインタラクティブなデモ（実際に UI 要素をハイライトするスポットライト形式）を追加することができます。
